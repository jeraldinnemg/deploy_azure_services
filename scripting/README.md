# acd-scripting
Para clonar el repo, tienen dos maneras. La facil y la 'complicada'. (GUI vs consola)

### 1. Instalar Git For Windows
Todo arranca con instalar 'Git for windows' https://gitforwindows.org/

### 2. Opcion GUI, Github desktop (Easy way)

<font color="#2DB757">✔</font>Instalan [Github Desktop](https://docs.github.com/en/desktop/contributing-and-collaborating-using-github-desktop/adding-and-cloning-repositories/cloning-and-forking-repositories-from-github-desktop), clickean 'clonar'

<font color="#2DB757">✔</font>Clone, and that's it. 

You will be able to manage your repo, commits and pushs if you'd like, from it through the GUI.

### 3. Console Option (complicated way):

<font color="#2DB757">✔</font>Abren la consola de git for windows
<font color="#2DB757">✔</font>git config --global user.name A2101365 (Su usuario de github)
<font color="#2DB757">✔</font>git config --global user.email ariel.masciotta@gds.ey.com (su mail de github)

### https o por ssh

### https:

<font color="#2DB757">✔</font>git clone https://github.com/a2101365/acd-scripting.git

### ssh: 

<font color="#2DB757">✔</font> Crear SSH Keys ([Link](https://docs.github.com/en/authentication/connecting-to-github-with-ssh/generating-a-new-ssh-key-and-adding-it-to-the-ssh-agent))
<font color="#2DB757">✔</font>Navigate to folder where you want the folder to be created (cd /c/examplefolder)
<font color="#2DB757">✔</font>git clone git@github.com:a2101365/acd-scripting.git

### Notas:

De todas maneras logran lo mismo. Clonar el repo para poder crear branches, commitear codigo, y pushearlo.
